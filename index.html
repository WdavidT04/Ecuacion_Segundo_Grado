Ecuación de Segundo Grado

Definición.

Una ecuación de segundo grado​​ o ecuación cuadrática de una variable es una ecuación que tiene la expresión general: donde x es la variable, y a, b y c constantes; a es el coeficiente cuadrático, b el coeficiente lineal y c es el término independiente.

ax2+bx+c=0

Forma general de una ecuación de segundo grado

Discriminante y posibles soluciones

Por comodidad, resolveremos la ecuacion de tres formas distintas segun los valores de los coeficioentes b y c.

Se llama discriminante,Δ, a:

Δ = b 2-4ac

La discriminante nos permite conocer el tipo de soluciones que tiene la ecuación:

  • Si Δ > 0, hay dos soluciones reales distintas.
  • Si Δ = 0, hay una única solucion y es real.
  • Si Δ < 0, no hay soluciones reales.
